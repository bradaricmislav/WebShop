# Responsive Webshop

A **responsive frontend webshop** built with **HTML, CSS, and Vanilla JavaScript**.

## Features
- Dynamic product filtering by **category** and **platform**
- **Search bar** to quickly find products
- Interactive **shopping cart** with automatic total price calculation
- **responsive design** 

## Technologies Used
- HTML, CSS, JavaScript (Vanilla JS)
- DOM manipulation and event handling
- CSS Flexbox and Grid for layout
- Local storage for cart persistence 

**Note:** To run the webshop correctly, the project should be opened using a **live server**.
